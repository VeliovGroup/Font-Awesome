Отправить (пул реквест) Pull Requests

Нашли способ решения ошибки в Font Awesome? Хотите добавить новый функционал? Помните об:
  1. Все pull request'ы отправляйте в ветку *-wip.
  2. Pull request'ы отправленые в ветку master будут откланены.
  3. После выполнения пунктов выше отправьте pull request.
